# 3.定时

定时有两种 1是setInterval() ：按照指定的周期（以毫秒计）来调用函数或计算表达式。方法会不停地调用函数，直到 clearInterval() 被调用或窗口被关闭。 2是setTimeout() ：在指定的毫秒数后调用函数或计算表达式。

[关键字: NAN意思是不是数字 setTimeout是js中的关键字 定时 return 非必须选项 return后面的代码统统不会执行，意思是就此跳出了 参数 非必须选项 一一对应 注意顺序对应]
